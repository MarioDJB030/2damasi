Raspberry Pi

Configuración Regional (Set Country)

 

En esta primera fase del asistente de bienvenida (Welcome Wizard), se establecen los parámetros de localización del sistema. Seleccionamos el País (Spain), el Idioma (Spanish) y la Zona Horaria (Madrid). Este paso es crucial no solo para la interfaz visual, sino porque configura la distribución del teclado correcta; si esto no se hace bien al inicio, caracteres como la "ñ" o los símbolos no coincidirán con las teclas físicas.

 

Creación de Usuario (Create User)

 

A continuación, el sistema solicita la creación de una cuenta de usuario administrador, ya que, por seguridad, las versiones recientes de Raspberry Pi OS ya no utilizan el usuario "pi" por defecto. Siguiendo las instrucciones de la práctica, hemos configurado el nombre de usuario como aulaateca26 y hemos establecido la misma cadena de caracteres para la contraseña. Esto garantiza el acceso estandarizado a los equipos del aula.

 

Selección de Red (Select WiFi Network)

 

El dispositivo realiza un escaneo de las redes inalámbricas disponibles en el entorno (SSID), detectando tanto bandas de 2.4GHz como de 5GHz. Seleccionamos la red específica del centro, denominada Wifi_AULA_ATECA, e introducimos las credenciales correspondientes. Conectarse en este paso es fundamental para que el asistente pueda descargar las actualizaciones y paquetes de idioma en las fases siguientes.

 

Elegir Navegador (Browser)

 

El sistema nos ofrece la opción de elegir el navegador web predeterminado. Aunque Firefox es una opción válida, seleccionamos Chromium siguiendo las directrices de la práctica. Chromium es la versión de código abierto en la que se basa Google Chrome y suele ser el estándar recomendado en Raspberry Pi OS por su optimización para la arquitectura ARM del dispositivo.

 

Actualización de Software (Update Software)

 

Una vez conectados a internet, el asistente busca actualizaciones en los repositorios oficiales. El proceso consta de varias partes: primero verifica la lista de paquetes, luego descarga e instala los paquetes de idioma (para poner el sistema completamente en español) y finalmente descarga las actualizaciones del sistema operativo. Cabe destacar que este último paso tuvo una duración considerable, debido probablemente al volumen de actualizaciones acumuladas y a la velocidad de escritura de la tarjeta microSD.

 

 

Configuración Completada (Setup Complete)

 

El asistente nos confirma que el proceso de configuración inicial ha finalizado con éxito. En este punto, todos los cambios de localización, usuario y red han sido aplicados. El sistema está listo para ser utilizado, aunque generalmente se recomienda un reinicio (Reboot) para asegurar que todos los servicios y actualizaciones del kernel se carguen correctamente.

Prueba de Velocidad (Speed Test)

 

Para verificar la estabilidad y el rendimiento de la conexión a internet configurada anteriormente, abrimos el navegador Chromium y realizamos un test de velocidad (Speed Test). Esta prueba nos permite medir la latencia (ping), así como las velocidades de bajada y subida en tiempo real, confirmando que la antena Wi-Fi de la Raspberry Pi está funcionando correctamente dentro de la red del aula.

 

Mac Mini

Selección de Idioma y Región

 

Al iniciar el Mac mini por primera vez, el Asistente de Configuración (Setup Assistant) nos recibe solicitando el idioma principal para el sistema operativo. Seleccionamos Español, seguido de la selección del país o región (España). Esta elección no solo traduce la interfaz, sino que preconfigura formatos de fecha, moneda y unidades métricas adecuados para la región.

Personalización del Tratamiento

 

macOS incluye opciones de inclusión lingüística. En esta pantalla, el sistema nos pregunta cómo queremos ser tratados gramaticalmente (femenino, masculino o neutro). Esto permite que el sistema operativo adapte los textos, alertas y mensajes de bienvenida para dirigirse al usuario de la manera más natural y correcta posible según su preferencia.

Confirmación de Idioma y Teclado

 

Basándose en las selecciones anteriores, el sistema presenta un resumen de la configuración de entrada ("Lenguas escritas y habladas"). Aquí confirmamos que el Idioma preferido es Español y verificamos que la fuente de entrada (distribución del teclado) corresponda a Español - ISO. Es vital revisar esto para asegurar que la posición de las teclas físicas coincida con los caracteres en pantalla.

Accesibilidad

 

Apple integra las opciones de accesibilidad desde el primer momento. El asistente nos ofrece configurar adaptaciones para necesidades visuales, motrices, auditivas y cognitivas (como VoiceOver o Zoom). Aunque para esta práctica seleccionamos "Ahora no" para proceder con una instalación estándar, es importante notar que estas herramientas están disponibles desde el arranque para garantizar que cualquier usuario pueda configurar el equipo.

Conexión a Red Wi-Fi

 

El dispositivo escanea las redes disponibles. Seleccionamos la red del aula Wifi_AULA_ATECA e introducimos la contraseña. En el entorno de Apple, conectarse a internet en este paso es fundamental no solo para actualizaciones, sino para la activación del dispositivo contra los servidores de Apple y la sincronización automática de la hora.

 

Asistente de Migración

 

El sistema nos ofrece la posibilidad de transferir información (datos, cuentas, configuraciones) desde otro Mac, desde una copia de seguridad de Time Machine o incluso desde un PC con Windows. Dado que estamos realizando una instalación limpia (clean install) para el aula, seleccionamos la opción de "Ahora no" o configurar como un nuevo Mac.

Iniciar Sesión con Apple ID

 

Se nos solicita iniciar sesión con un ID de Apple para vincular servicios como iCloud, App Store y Buscar mi Mac. Siguiendo las instrucciones de la práctica para un equipo de uso compartido en un aula, omitimos este paso (selecionando "Configurar más tarde"). Esto evita que el equipo quede vinculado a una cuenta personal y permite una gestión más genérica.

 

Términos y Condiciones

 

Se presentan los Términos y Condiciones de la licencia de software de macOS (EULA). Este es un contrato legal vinculante entre el usuario y Apple que detalla los derechos de uso, garantías y responsabilidades. Es necesario hacer clic en Aceptar para poder continuar con la instalación y uso del sistema operativo.

Creación de la Cuenta del Ordenador

 

Al no usar iCloud, creamos una cuenta local de administrador. Siguiendo la normativa del ejercicio, establecemos el nombre de cuenta y nombre completo como aulaateca26 y configuramos la contraseña idéntica. Esta cuenta será la propietaria del directorio "Home" y tendrá permisos para instalar aplicaciones y modificar ajustes del sistema.

Configuración de Zona Horaria

 

El sistema nos pide confirmar nuestra ubicación para ajustar la zona horaria. Al tener conexión a internet y servicios de localización (si se activaron), suele detectar automáticamente Madrid (CET). Confirmamos esta selección para asegurar que los registros del sistema (logs) y la navegación web segura (certificados SSL) funcionen correctamente con la hora actual.

 

Tiempo de Uso (Screen Time)

 

El asistente nos informa sobre la funcionalidad "Tiempo de Uso", que permite monitorizar cuánto tiempo se pasa frente a la pantalla y establecer límites para aplicaciones específicas. El sistema ofrece configurarlo ahora o más tarde; generalmente se nos muestra un resumen de sus capacidades. Continuamos el proceso aceptando la configuración predeterminada.

 

 

Bienvenida y Fin de la Configuración

 

Finalmente, el asistente concluye la configuración. Tras esto, aparece la pantalla de "Bienvenida" y, tras unos segundos de carga, se muestra por primera vez el escritorio (Desktop) y el Dock, indicando que el Mac mini está completamente operativo y listo para trabajar.
