Configuración inicial de la Jetson Nano

Mario De Juana Baeza
diciembre 4, 2025
Updated on diciembre 4, 2025

 

1. Introducción y Objetivos

El objetivo de este documento es detallar el procedimiento de configuración inicial y puesta en marcha del kit de desarrollo NVIDIA Jetson Nano.

Este proceso abarca desde la conexión física de los periféricos hasta la configuración del software de primer inicio (first boot). El equipo opera bajo una versión de Ubuntu (Linux), personalizada por NVIDIA (JetPack), que requiere una serie de pasos de configuración básica para dejar el sistema listo para su uso.

Paso 1: Conexiones Físicas

Para empezar, realizamos las conexiones físicas necesarias en la placa NVIDIA Jetson Nano, tal como se observa en la imagen. Esto incluye:

  • Cable HDMI: Conectado para la salida de vídeo hacia el monitor.

  • Periféricos USB: Se han conectado el teclado y el ratón a los puertos USB.

  • Fuente de Alimentación: Conectada para proporcionar energía a la placa.

Nota: Creo que mi compañero tuvo que instalar antes el disco nvme.

Paso 2: Aceptación de Licencias

Una vez que el sistema ha arrancado, aparecerá el asistente de Configuración del Sistema (System Configuration).

La primera pantalla nos solicita revisar y aceptar los acuerdos de licencia de usuario final de NVIDIA. Para continuar:

  1. Marcamos la casilla inferior izquierda que indica: "I accept the terms of these licenses" (Acepto los términos de estas licencias).

  2. Hacemos clic en el botón "Continue".

Paso 3: Selección del Idioma

A continuación, aparecerá la pantalla de bienvenida ("Le damos la bienvenida") donde debemos configurar el idioma de la interfaz del sistema operativo.

  1. En la lista de idiomas disponibles, localizamos y seleccionamos Español.

  2. Una vez seleccionado, hacemos clic en el botón Continuar situado en la esquina inferior derecha.

Paso 4: Disposición del Teclado

En esta pantalla debemos configurar la distribución del teclado para que coincida con el periférico físico conectado.

  1. En la columna de la izquierda, buscamos y seleccionamos Spanish (Español).

  2. Verificamos que en la columna de la derecha también esté seleccionada la opción Spanish (o la variante específica que corresponda a tu teclado).

  3. Podemos usar el campo de texto inferior ("Escriba aquí para probar el teclado") si queremos verificar que caracteres especiales como la "ñ" o las tildes funcionan correctamente.

  4. Hacemos clic en Continuar.

 

Paso 5: Configuración de Red Inalámbrica (Wi-Fi)

Para dotar de conectividad al equipo, seleccionamos la red inalámbrica disponible.

  1. En la lista de redes detectadas, seleccionamos la red deseada (en este caso, Wifi_AULA_ATECA).

  2. Aparecerá una ventana emergente solicitando la autenticación ("Authentication required by Wi-Fi network").

  3. Introducimos la contraseña de seguridad de la red.

  4. Hacemos clic en el botón Connect (Conectar) para establecer la conexión.

Paso 6: Configuración de la Ubicación

En esta pantalla configuraremos la zona horaria del sistema para asegurar que la fecha y la hora sean correctas.

  1. El sistema intentará detectar tu ubicación automáticamente si estás conectado a internet. En el mapa, asegúrate de que el marcador esté situado sobre tu región actual.

  2. Verifica que en el campo de texto inferior aparezca Madrid (o la ciudad correspondiente a tu zona horaria).

  3. Una vez confirmada la ubicación, hacemos clic en el botón Continuar para avanzar.

Paso 7: Configuración de Usuario y Equipo

En este paso crearemos la cuenta de usuario principal y definiremos el nombre del dispositivo en la red.

  1. Rellenamos los campos con la información requerida:

    • Su nombre: Introducimos el nombre para identificar al usuario (en la imagen: aulaateca26).

    • El nombre de su equipo: Este será el identificador visible en la red (ejemplo: aulaateca26-desktop).

    • Nombre de usuario: El nombre corto que usará el sistema internamente (ejemplo: aulaateca26).

    • Contraseña: Establecemos y confirmamos una contraseña segura para el administrador.

  2. En las opciones de inicio de sesión, seleccionamos "Iniciar sesión automáticamente". Esto permitirá que la Jetson Nano arranque directamente al escritorio sin detenerse a pedir la contraseña.

  3. Hacemos clic en Continuar.

Paso 8: Tamaño de la Partición de Aplicaciones (APP Partition Size)

En este paso, el sistema nos solicita definir el tamaño que tendrá la partición principal donde se alojarán el sistema operativo y las aplicaciones.

  1. El asistente calculará y rellenará automáticamente el campo con el tamaño máximo disponible en el disco (en este caso, 117782 MB).

  2. Se recomienda mantener este valor por defecto para aprovechar toda la capacidad de almacenamiento del dispositivo.

  3. Hacemos clic en Continuar.

Paso 9: Instalación del Navegador Chromium

Finalmente, el asistente de configuración nos ofrecerá la opción de instalar el navegador web Chromium para tener acceso a internet desde el escritorio.

  1. Asegúrate de tener seleccionada la opción "Install Chromium Browser".

  2. El sistema descargará e instalará el paquete automáticamente. Verás una ventana de progreso con el texto "Installing Chromium Browser, please wait...".

  3. Una vez que aparezca el mensaje confirmando que la instalación ha finalizado ("chromium installation is finished"), hacemos clic en el botón Close de la ventana emergente.

  4. Para terminar, hacemos clic en el botón Continuar.

 

Paso 10: Aplicación de Configuración y Arranque

Una vez completados todos los pasos anteriores e instalado el navegador, el sistema procederá a aplicar los cambios.

  1. Veremos una pantalla con una barra de progreso bajo el título "Configuración del sistema".

  2. El mensaje indicará "> Configurando el hardware...".

  3. Este proceso es automático y puede tardar unos minutos. Es importante no apagar ni desconectar la Jetson Nano durante este proceso.

  4. Al finalizar, el sistema cargará el entorno de escritorio de Ubuntu.

Paso 11: Asistente de Bienvenida (Post-Reinicio)

Tras el reinicio automático, el sistema cargará el entorno gráfico y mostrará un último asistente de bienvenida ("Welcome") para terminar de configurar preferencias de usuario.

  1. Cuentas en Línea (Online Accounts): Aparecerá una pantalla para conectar cuentas de Google, Microsoft, etc.

    • Para esta configuración, no vincularemos ninguna cuenta. Hacemos clic en el botón Skip (Omitir) situado en la esquina superior derecha.

  2. Ubuntu Pro:

    A continuación, el sistema ofrecerá habilitar "Ubuntu Pro".

    • Seleccionamos también la opción de Skip (Omitir) o Next (Siguiente) sin activar el servicio para continuar con la configuración estándar.

Paso 12: Actualización del Sistema

Una vez en el escritorio, aparece automáticamente el gestor de actualizaciones (Software Updater) detectando que hay versiones más recientes del software desde el lanzamiento de Ubuntu 22.04.

  1. Aparecerá una ventana indicando: "Updated software has been issued since Ubuntu 22.04 was released. Do you want to install it now?".

  2. Hacemos clic en el botón Install Now (Instalar ahora) para asegurar que el sistema tenga los últimos parches de seguridad y controladores.

  3. Nos pide la contraseña de usuario que creamos en el Paso 7 para autorizar la instalación.

Paso 13: Conexión de Teclado y Ratón Inalámbrico (Bluetooth)

Para finalizar la configuración del hardware y mejorar la ergonomía, procederemos a enlazar el teclado y el ratón inalámbricos a través de Bluetooth.

  1. Accedemos al menú de Configuración (Settings) del sistema y seleccionamos la pestaña Bluetooth en la barra lateral izquierda.

  2. Aseguramos que el interruptor de Bluetooth (arriba a la derecha) esté activado.

  3. Ponemos nuestros periféricos en modo de emparejamiento.

  4. En la lista de Dispositivos (Devices), seleccionamos el teclado y el ratón para conectarlos.

    • En este caso, conectamos los dispositivos "POP Icon Keys" (Teclado) y "Logi POP Mouse" (Ratón).

  5. Una vez que el estado cambie a "Connected" (Conectado), los periféricos estarán listos para usarse sin cables.

Con este último paso, el kit de desarrollo NVIDIA Jetson Nano está completamente configurado, actualizado y listo para iniciar.
